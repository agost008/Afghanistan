Afghanistan Demo - versione locale

Questa versione si apre senza server.

Come usarla:
1. Estrai lo ZIP.
2. Entra nella cartella public.
3. Apri index.html con doppio clic oppure con VS Code + browser.

File principali:
- public/index.html
- public/article.html
- public/styles.css
- public/script.js
- public/article.js
- public/data.js

Nota:
Le pagine articolo si aprono dai link interni della home. Non serve Java e non serve avviare nessun server.
